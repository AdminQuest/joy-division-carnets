Carnet de notes préparatoire

Ce site réunit des billets de travail de Fabrice Ribet autour de Joy Division. Ils préfigurent la publication du livre Joy Division, le son de l’éternel : chaque note teste un angle, une scène, une source ou une prudence historiographique.

Index thématique — version du 14 mai 2026.

Cette page n’est pas un flux chronologique. Les articles sont classés par thèmes. Pour éviter les doublons, seule la version française apparaît dans l’index principal ; la version anglaise reste accessible depuis chaque article.
